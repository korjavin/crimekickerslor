# Naming Variants

Names drifted across prototypes. This page is the canonical map from any spelling to
the primary page. (Each variant is also set as an Obsidian `alias` on its page, so
`[[Phoboman]]`, `[[Fuboman]]`, etc. all resolve.)

## The team

| Variant | Meaning |
| --- | --- |
| **Crime Kickers** | The canonical team / universe name. |
| Stealth Kickers | Prototype alias (the [[Stealth Kickers (Tactical Stealth)|stealth game]]). |
| Shadow Kickers | Repo/title alias (the [[Shadow Kickers (2D Platformer)|2D platformer]]). |
| Stealth Action Squad | In-game working team name in *Stealth Kickers*. |

> Note: *Fubomen* is the spelling used in the source registry
> (`crimekickers-init.md`); *Pho-boman* is the Hub spelling and is used here as
> primary because it carries the "pho soup" pun. *Tiebi* chosen as primary spelling;
> *Tiebe* is the most common in the Hub.
