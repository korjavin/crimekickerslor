# Shadow Kickers (2D Platformer)

A 2D stealth-puzzle platformer — full title **"Crime Kickers vs The Bad Guys."**
Inspired by real-time tactics games (*Shadow Tactics*, *Commandos*) reimagined as a
side-scroller. Single-file HTML/JS game.

## Core mechanic

Switch (keys 1–4) between four heroes mid-level; each level has obstacles only a
specific hero can pass, so you must use the whole squad. *"No single character can
pass alone."*

## Roster in this build

> This is an **early build that predates [[Pho-boman]]'s inclusion** in the team. The
> canon heroes featured here are:

- **Windman** (blue): Gale Force wind gust (blow/stun enemies, raise wind platforms).
- **Teibi** (green): Micro Shift (shrink to fit crawl-gaps). Titan Shift (grow) noted
  but not implemented. → [[Tiebi]]
- **Primm** (purple): Phase (hold C to pass through purple walls) + Katana dash.

## Lore it contributes

- The title establishes **"Crime Kickers vs The Bad Guys"** and "Crime Kicker" as an
  achievement title (*"the ultimate Crime Kicker!"*).
- Confirms [[Windman]]'s **Disguise** idea and the **wind/phase/shrink/dash** ability
  set shared across games.
- Enemies: generic red [[The Bad Guys|"Bad Guys"]].

## Visual direction

Bright cartoon palette (blue sky → green grass), flat rectangles. A planned
"visual-style-polish" pass adds parallax, shading, shadows, particles, and screen
shake — depth without changing gameplay.
